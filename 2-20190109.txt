 * java source 작성

   - public static void main(String[] args)
      - main메소드는 하나만 선언할수있다
      - public static -> 수정자
      - void main -> 매소드타입 매소드명
      - (String[] args)  ->  매개변수타입 매개변수

 * 식별자 - 클래스명,메소드명,변수 등을 칭함, 대소문자구분
   - 클래스명: 파스칼표기법
   - 메소드명, 변수: 카멜표기법
   - 상수 : 모두 대문자
   - 패키지 : 모두 소문자
   - 화면GUI : 헝가리안 표기법

 * 상수(변하지않는값)선언은 final `변수타입 변수명` 으로 선언한다

 * 데이터 타입
   1.기본형타입(8개)
     - 수치형(6개) 정수형 < 실수형
       정수형 :byte(1) < short(2) < int(4) < long(8)
       실수형 : float(4) < double(8)
     - 문자형(1개) : char -> '' 문자하나
     - 논리형(1개) : boolean -> true, false

   2.참조형타입 : class타입, 배열타입 등
     - 변수에 직접적으로 값이 들어가는것이 아니라, 값을 참조할수있는 해쉬코드가 변수에 들어간다...
     - ex) int[] j = {1,2,3,4,5}
     - ex) String str = "값";
     - new를 통해 객체생성
     - 클래스명과 매소드명이 같을경우, 리턴타입이 존재하지 않는것은 생성자 클래스이다



 * 형변환 : 수치형 변환
    - 높은 타입의 값을 낮은 타입의 값으로 변환하는것을 casting이라고 하고 그 반대는 자동변환된다.
    - casting ex) k = (int)f;  //작은 타입 = (작은 타입)큰타입 
    - 자동변환 ex) k = f;

 * 연산자 : 산술연산자, 복합대입연산자, 증감연산자, 비교연산자, 비트연산자, 논리연산자
